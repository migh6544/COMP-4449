
# Steel Surface Defect Detection & Classification

## Overview
This project implements machine learning pipelines for detecting and classifying steel surface defects
using the NEU Surface Defect Database.

Two modeling pipelines are implemented:

1. Image Classification
   - Baseline CNN
   - EfficientNetB0 Transfer Learning
   - Stratified K-Fold Cross Validation

2. Object Detection
   - YOLOv8 detection pipeline using Ultralytics
   - Detection performance evaluation with confusion matrices

The goal is to build a reliable system for automated defect inspection in manufacturing environments.

---

## Dataset

Dataset: NEU Surface Defect Database

Classes:
- crazing
- inclusion
- patches
- pitted_surface
- rolled_in_scale
- scratches

Images are grayscale steel surface samples labeled with defect classes.

## Running the Classification Pipeline

Example:

```
python phase2_rigorous_cv_pipeline_ready.py     --train_dir data/train     --test_dir data/test     --run cv_transfer
```

This script performs:

- dataset loading
- stratified K-fold cross validation
- EfficientNet transfer learning
- evaluation metrics generation

Outputs include:

- confusion matrices
- model performance metrics
- training logs

---

## Running the Detection Pipeline

Example:

```
python phase2_yolo_detection_pipeline_ready_v4.py
```

This pipeline:

- prepares YOLO formatted datasets
- trains YOLOv8 detector
- evaluates detection accuracy
- generates training curves and confusion matrices
